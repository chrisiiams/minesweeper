# Minesweeper 

A classic Minesweeper game built with vanilla JavaScript.

Built as part of my portfolio-website to get hands-on with JavaScript fundamentals like 2D arrays, 
DOM manipulation, event handling and scope.

## How to Play
- Left click to reveal a cell
- Right click to place a flag 
- Reveal all non-mine cells to win!

## What I Learned
- Representing a grid as a 2D array
- DOM manipulation and dynamic element creation
- Event handling (onclick, oncontextmenu)
- Scope in JavaScript
- Recursive neighbour checking
